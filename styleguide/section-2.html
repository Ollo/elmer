2.2 - $selection

This color is used to manage the highlight color.

$selection
::selection, ::-moz-selection { background: $selection; }
<div style="background:#fff2a8; color: #999999; padding: 1rem;">$selection</div>
::selection, ::-moz-selection {
     background: $selection;
}
